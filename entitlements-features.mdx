### Unlocking Entitlement Features

Designed with developers in mind to provide fine-grained authorization, speed of deployment, and flexibility. Our proprietary Frontegg CALC technology enables powerful context-based authorization based on User-entitlements. The engine is divided into four distinctive areas that control a user's access to data:

[ Roles Enable access based on user-specific roles est. 3 min ](https://docs.frontegg.com/v2.0.2/docs/roles-1) [ Permissions Allocate Permissions to specific areas based on user Entitlements est. 3 min ](https://docs.frontegg.com/v2.0.2/docs/permissions-1) [ Features Create Entitlement-based access to specific Features est. 3 min ](https://docs.frontegg.com/docs/features)[ Plans Create Entitlement-based access to specific Feature Plans est. 3 min ](https://docs.frontegg.com/v2.0.2/docs/feature-bundles) [ Feature Flags Create rule-based Feature Flags est. 3 min ](https://docs.frontegg.com/docs/feature-flags)
